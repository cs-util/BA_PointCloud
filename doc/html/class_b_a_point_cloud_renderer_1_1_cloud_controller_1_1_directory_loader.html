Detailed Description

Use this loader, if you have several pointcloud-folders in the same directory and want to import all of them to your program. To import them, create a DirectoryLoader and press the "Load Directory" Button in the Editor (or call the function LoadAll).

Streaming Assets support provided by Pablo Vidaurre
